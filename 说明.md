# 扭一扭 — Hermes Agent 技能

像扭蛋机一样，随机扭一个抖音搞笑短视频出来！

```
hermes skills install https://raw.githubusercontent.com/chengnas/hermes-skill-niuyiniu/main/SKILL.md
```

## 这是什么

一个 Hermes Agent 的零 Token 技能。自动从公开抖音 API 拉取热门短视频，直接投送到你的聊天窗口，全程不消耗 AI Token。

## 两种玩法

### 🕐 定时投喂（推荐）

```bash
# 安装脚本
mkdir -p ~/.hermes/scripts
cp ~/.hermes/skills/niuyiniu/scripts/niuyiniu.sh ~/.hermes/scripts/
chmod +x ~/.hermes/scripts/niuyiniu.sh

# 每天 9:00 / 12:00 / 18:00 自动发
hermes cron create "0 9,12,18 * * *" \
  --name "扭一扭" \
  --script niuyiniu.sh \
  --no-agent \
  --deliver origin
```

### 🎮 手动触发

在聊天里输入 `/扭一扭`，随时扭一个。

需要改源码注册 slash 命令 + 配置 hook，详见 SKILL.md 里的方案二。

## 原理

```
用户触发 (/扭一扭 或 cron)
    → Gateway hook 拦截
    → Shell 脚本 curl API
    → 输出 MEDIA:/tmp/niuyiniu.mp4
    → 视频送达聊天窗口
    → 0 AI Token！
```

## 视频来源

`http://api.qemao.com/api/douyin/` — 免费公开 API，无需 Key
